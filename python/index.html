Variables y tipos de datos

Se declaran sin tipo explícito. Tipos básicos incluyen int, float, str, list, dict.

x = 5          # int
y = 10.5       # float
nombre = "Ramses"  # str
frutas = ["manzana", "banana"]  # list

Funciones

Bloques reutilizables de código.

def saludar(nombre):
    return "Hola " + nombre

print(saludar("Ramses"))  # "Hola Ramses"

Estructuras de control

Controlan el flujo del programa: condicionales y bucles.

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor")

for i in range(5):
    print(i)

Evita que el programa se detenga ante fallos.

Módulos e importaciones

Permiten dividir el código en archivos reutilizables.

# archivo saludo.py
def saludar():
    return "Hola"

# archivo principal.py
from saludo import saludar
Fuente: [W3Schools](https://www.w3schools.com/python/python_modules.asp)

JSON

Formato ligero para intercambio de datos entre sistemas.

import json

usuario = {
    "nombre": "Ramses",
    "edad": 35
}

json_str = json.dumps(usuario)  # a texto
obj = json.loads(json_str)       # a objeto
Fuente: [W3Schools](https://www.w3schools.com/python/python_json.asp)
