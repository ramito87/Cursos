JavaScript: Guía Visual Completa

Tarjetas explicativas para dominar JavaScript desde sus fundamentos hasta técnicas modernas. Ideal para consulta rápida y ritual técnico.

Historia de JavaScript

Creado en 1995 por Brendan Eich en Netscape. Evolucionó con ECMAScript y se convirtió en el estándar web.

// Nombres iniciales
Mocha → LiveScript → JavaScript

// Estándares clave
ES1 (1997), ES5 (2009), ES6 (2015)
Fuente: [GeeksforGeeks](https://www.geeksforgeeks.org/javascript/history-of-javascript/), [W3Schools](https://www.w3schools.com/js/js_history.asp), [Dev.to](https://dev.to/saartwito/the-evolution-of-javascript-from-inception-to-modern-day-282k)

Variables y alcance

Se declaran con var, let y const. Cada una tiene reglas de alcance distintas.

var x = 5; // global o función
let y = 10; // bloque
const z = 15; // constante
Fuente: [W3Schools](https://www.w3schools.com/js/js_scope.asp), [FreeCodeCamp](https://www.freecodecamp.org/news/scope-in-javascript-global-vs-local-vs-block-scope/)

Tipos de datos

JavaScript tiene 8 tipos básicos: string, number, boolean, null, undefined, symbol, bigint, object.

let nombre = "Ramses"; // string
let edad = 35;         // number
let activo = true;     // boolean
let datos = { nombre, edad }; // object
Fuente: [Programiz](https://www.programiz.com/javascript/data-types), [W3Schools](https://www.w3schools.com/js/js_datatypes.asp)

Manipulación del DOM

El DOM permite modificar HTML desde JavaScript.

const titulo = document.querySelector("h1");
titulo.textContent = "Nuevo título";
Fuente: [JavaScriptTutorial.net](https://www.javascripttutorial.net/javascript-dom/), [FreeCodeCamp](https://www.freecodecamp.org/espanol/news/el-dom-de-javascript-un-tutorial-practico/)

Funciones y arrow functions

Las funciones son bloques reutilizables. Las flechas simplifican la sintaxis.

// Tradicional
function saludar(nombre) {
  return "Hola " + nombre;
}

// Flecha
const saludar = nombre => "Hola " + nombre;
Fuente: [W3Schools](https://www.w3schools.com/Js/js_arrow_function.asp), [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/Arrow_functions)

Estructuras de control

Controlan el flujo del programa: condicionales y bucles.

if (edad >= 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor");
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}
Fuente: [GeeksforGeeks](https://www.geeksforgeeks.org/javascript/control-statements-in-javascript/), [Platzi](https://platzi.com/blog/guia-completa-de-estructuras-de-control-en-javascript-if-else-if-else-y-switch/)

Eventos y listeners

Permiten reaccionar a acciones del usuario.

document.getElementById("btn")
  .addEventListener("click", () => {
    alert("¡Click!");
  });
Fuente: [W3Schools](https://www.w3schools.com/js/js_htmldom_eventlistener.asp), [MDN](https://developer.mozilla.org/es/docs/Web/API/EventTarget/addEventListener)

Promesas y async/await

Manejo moderno de operaciones asíncronas.

// Promesa
fetch("api/datos")
  .then(res => res.json())
  .then(data => console.log(data));

// Async/await
async function cargar() {
  const res = await fetch("api/datos");
  const data = await res.json();
  console.log(data);
}
Fuente: [JavaScript.info](https://javascript.info/async-await), [GeeksforGeeks](https://www.geeksforgeeks.org/javascript/callbacks-vs-promises-vs-async-await/)

Manejo de errores

Evita que el programa se detenga ante fallos.

try {
  ejecutar();
} catch (error) {
  console.error("Error:", error.message);
}
Fuente: [MDN](https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Statements/try...catch), [JavaScript.info](https://es.javascript.info/try-catch)

Módulos e importaciones

Permiten dividir el código en archivos reutilizables.

// archivo saludo.js
export function saludar() {
  return "Hola";
}

// archivo principal.js
import { saludar } from "./saludo.js";
Fuente: [W3Schools](https://www.w3schools.com/js/js_modules.asp), [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/import)

Arrays

Colecciones ordenadas de elementos. Se accede por índice.

let frutas = ["manzana", "banana", "uva"];
console.log(frutas[0]); // "manzana"

// Métodos comunes
frutas.push("naranja");     // agregar
frutas.pop();               // eliminar último
frutas.includes("banana");  // true
Los arrays pueden contener cualquier tipo de dato, incluso otros arrays u objetos.

Objetos

Estructuras clave-valor que representan entidades.

let persona = {
  nombre: "Ramses",
  edad: 35,
  saludar: function() {
    return "Hola " + this.nombre;
  }
};

console.log(persona.saludar()); // "Hola Ramses"
Los objetos son la base de la programación orientada a objetos en JS.

Clases

Plantillas para crear objetos con propiedades y métodos.

class Animal {
  constructor(nombre) {
    this.nombre = nombre;
  }
  hablar() {
    console.log(this.nombre + " hace ruido.");
  }
}

const perro = new Animal("Firulais");
perro.hablar(); // "Firulais hace ruido."
Las clases fueron introducidas en ES6 como azúcar sintáctico sobre prototipos.

JSON

Formato ligero para intercambio de datos entre sistemas.

let usuario = {
  nombre: "Ramses",
  edad: 35
};

let json = JSON.stringify(usuario); // a texto
let obj = JSON.parse(json);         // a objeto
JSON es usado en APIs, almacenamiento y configuración.

localStorage

Almacena datos en el navegador de forma persistente.

// Guardar
localStorage.setItem("nombre", "Ramses");

// Leer
let nombre = localStorage.getItem("nombre");

// Eliminar
localStorage.removeItem("nombre");
Ideal para guardar preferencias, sesiones o progreso local.

Consumo de APIs

JavaScript puede conectarse a servicios externos usando fetch.

fetch("https://api.example.com/datos")
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error("Error:", err));
Usa headers y métodos HTTP para POST, PUT, DELETE según la API.
